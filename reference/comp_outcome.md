# Example composite outcome data used for Figure 12

Example composite outcome data used for Figure 12

## Usage

``` r
comp_outcome
```

## Format

A data frame with 1800 rows and 6 variables

- usubjid:

  Subject ID

- visit:

  Visit

- trtn:

  Treatment arms in numeric type

- trt:

  Treatment arms in character type

- brcatn:

  Category of composite outcome in numeric type

- brcat:

  Category of composite outcome in character type
