# Example scatterplot data used for Figure 11

Example scatterplot data used for Figure 11

## Usage

``` r
scatterplot
```

## Format

A data frame with 500 rows and 2 columns:

- bdiff:

  Simulated difference in incremental probabilities for active/control
  effects and outcome "Benefit"

- rdiff:

  Simulated difference in incremental probabilities for active/control
  effects and outcome "Risk"
