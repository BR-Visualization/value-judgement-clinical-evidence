# Criterion Weights for MCDA

A named numeric vector of stakeholder-elicited weights for each
criterion in the
[`mcda_data`](https://pkgdown.r-lib.org/reference/mcda_data.md) example
dataset. Weights sum to 1.

## Usage

``` r
data(weights)
```

## Format

A named numeric vector with 5 elements:

- Benefit 1:

  0.30

- Benefit 2:

  0.20

- Benefit 3:

  0.10

- Risk 1:

  0.30

- Risk 2:

  0.10

## Details

Example criterion weights for MCDA scoring
