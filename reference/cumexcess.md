# Example cumulative excess plot data used for Figure 13

Example cumulative excess plot data used for Figure 13

## Usage

``` r
cumexcess
```

## Format

A data frame with 880 rows and 10 columns:

- eventtime:

  Simulated event times

- diff:

  Simulated difference in active/control effects

- obsv_duration:

  Duration of observational period

- obsv_unit:

  Unit length of observational period

- outcome:

  Specifies Benefit/Risk

- eff_diff_lbl:

  Label for effect difference

- n:

  Number of subjects

- effect:

  Specifies active/control effect

- eff_code:

  0/1 depicting control/active effects

- subjects:

  Specifies number of subjects at a given time
